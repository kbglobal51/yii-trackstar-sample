安装EasyTalk - 目录检查

1、config.inc.php 文件可写 文件不可写或不存在
2、include/cache 目录可写 目录不可写或不存在
3、attachments/head 目录可写 目录不可写或不存在
4、attachments/usertemplates 目录可写 目录不可写或不存在
5、attachments/photo 目录可写 目录不可写或不存在
6、templates/cache 目录可写 目录不可写或不存在
7、admin/backup 目录可写 目录不可写或不存在
8、admin/templates/cache 目录可写 目录不可写或不存在
9、admin/themetemp 目录可写 目录不可写或不存在

目录检查

如果您的主机是linux，请确保左侧目录可写，如果您的主机是windows的可以直接点击下一步！
